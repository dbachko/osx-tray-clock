# osx-tray-clock

> OSX tray clock and calendar app


## Dev

```
$ npm install
$ ./node_modules/.bin/electron-rebuild
```

### Run

```
$ npm start
```

### Build app

```
$ npm run build:osx
```

### Create .dmg installation file

```
$ npm run pack:osx
```

Builds the app for OS X using [electron-packager](https://github.com/maxogden/electron-packager).
